# 我的世界使用好朋友联机工具联机教程

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **我的世界（Minecraft）**。

## 一、联机前准备

在开始之前，建议先确认以下内容：

### 1. 双方已安装好朋友联机工具

- 房主和好友都需要安装并登录好朋友联机工具。
- 建议使用最新版，避免因为版本过旧导致连接失败。

### 2. 双方进入同一个联机房间

- 房主创建房间。
- 好友加入同一个房间。
- 进入同一房间后，双方才能通过虚拟局域网互联。

### 3. 确认我的世界版本一致

- 如果是 **Java 版**，双方游戏版本要一致。
- 若使用了 Forge、Fabric、NeoForge 等模组环境，也需要保持一致。
- 如果装了整合包，建议整包保持一致。

## 二、Java 版：通过“对局域网开放”联机

这是最常见、最简单的方式，适合已经有单人存档的情况。

### 1. 房主操作

房主先进入游戏和存档，然后：

1. 进入一个单人世界。
2. 按下 `Esc` 打开菜单。
3. 点击 **对局域网开放**。
4. 根据需要设置游戏模式和作弊权限。
5. 点击确认后，游戏会显示一个局域网端口，例如 `xxxxx`。

### 2. 获取房主虚拟 IP

在好朋友联机工具房间里，查看房主对应的虚拟局域网 IP。

连接地址一般写成：

`房主虚拟IP:端口`

例如：

`10.0.0.8:51234`

### 3. 好友加入

好友打开我的世界后：

1. 进入 **多人游戏**。
2. 点击 **直接连接** 或 **添加服务器**。
3. 输入房主的虚拟 IP 与端口。
4. 点击加入游戏。

## 三、Java 版：通过独立服务端联机

如果你使用的是独立服务端，步骤会更简单一些。

### 1. 房主开启服务端

- 在本机启动 Minecraft 服务端。
- 确认服务端正常运行，默认端口通常为 `25565`。

### 2. 好友连接方式

好友在多人游戏中输入：

`房主虚拟IP:25565`

如果服务端改了端口，就使用实际端口号。

## 四、基岩版联机思路

如果你使用的是 **基岩版**：

- 也可以通过局域网方式联机；
- 前提是双方已加入同一个好朋友联机工具房间；
- 房主开启世界联机后，好友通常能在局域网会话中看到；
- 若看不到，也可以优先考虑使用支持基岩版的独立服务端方式。

## 五、常见问题处理

### 1. 搜不到世界

如果好友看不到房主的世界：

- 确认双方已进入同一个联机工具房间。
- 确认房主确实点击了 **对局域网开放**。
- 确认输入的是 **虚拟 IP + 端口**，不是本地 127.0.0.1。
- 尝试关闭系统防火墙后再测试。

### 2. 连得上工具，进不去游戏

这种情况通常和以下内容有关：

- 游戏版本不一致。
- MOD / 整合包不一致。
- 服务端与客户端模组要求不一致。
- 房主网络波动较大。

### 3. 延迟高、容易掉线

建议尝试：

- 尽量使用稳定网络；
- 减少后台下载和占用带宽的软件；
- 在工具里尝试更稳定的线路或中转模式；
- 服务端玩家较多时，适当降低视距和实体负载。

## 六、推荐使用方式

### 1. 轻度游玩

如果只是和 1~3 个朋友临时联机，推荐：

- 房主进单人世界；
- 使用 **对局域网开放**；
- 好友通过 **虚拟 IP + 端口** 连接。

### 2. 长期存档

如果是多人长期游玩，推荐：

- 直接搭建独立服务端；
- 所有人通过好朋友联机工具的虚拟 IP 连接；
- 这样会比反复开局域网更稳定。

## 七、总结

我的世界使用好朋友联机工具联机时，最关键的是：

1. 双方先进入同一个联机工具房间；
2. 房主开放局域网或启动服务端；
3. 好友使用 **房主虚拟 IP + 端口** 连接。

如果连接失败，优先检查 **联机房间、IP 地址、端口、版本和模组一致性**。
