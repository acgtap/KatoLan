# 我的世界使用好朋友联机工具联机教程

![我的世界联机界面](/help-img/mc.png)

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **我的世界（Minecraft）**，适用于临时开黑、长期存档和模组联机。

## 一、联机前准备

### 1. 安装联机工具和启动器

建议房主和好友都提前准备好：

- 好朋友联机工具
- Minecraft 启动器，例如 PCL2
- 对应版本的游戏本体
- 如需联机模组，还要准备一致的模组环境

![前期准备](/help-img/97d3405961c046d98a098a79348f4946.png)

### 2. 下载并安装好朋友联机工具

- 从官网下载压缩包并完成安装。
- 如果浏览器提示风险，通常需要手动选择保留该文件。
- 如果安装时被系统安全中心拦截，需要先恢复或加入白名单。

![下载提示](/help-img/7d4a8d5b2b1841bba2c3de480c0f09c1.png)

![安装提示](/help-img/97f0ba7682f74fc49316afa35d1f4c2d.png)

### 3. 创建或加入联机房间

进入好朋友联机工具后，最常用的是两个功能：

- **创建房间**：由一名玩家充当房主创建房间。
- **加入房间**：其他玩家输入相同房间号加入。

![联机工具主页](/help-img/软件首页.png)

![房间列表](/help-img/好友列表.png)

进入同一房间后，等待几秒让延迟稳定，再启动游戏会更稳。

## 二、PCL2 启动器和联机 Mod 准备

如果你是 Java 版，并且需要更灵活的联机方式，建议提前准备好版本和联机模组。

### 1. 下载游戏和加载器

在 PCL2 中选择需要的 Minecraft 版本，建议同时下载对应的模组加载器，例如 `Forge`。

![下载游戏与加载器](/help-img/ed415365765c4b4685e41818c4d5218c.png)

如果一开始没装加载器，也可以后面在手动安装界面补装。

![手动安装 Forge](/help-img/c20e603fc822431a9b520de64c6472ed.png)

### 2. 安装联机 Mod

如果你使用的是局域网联机增强方案，可以在 Mod 页面搜索并安装 **自定义局域网联机** 这类联机 Mod。

- Mod 版本要和游戏版本一致。
- 下载目录要放到当前实例的 `mods` 文件夹。
- 如果没有 `mods` 文件夹，通常说明还没有安装加载器。

![安装联机 Mod](/help-img/9d937f80c1cf4d8daa7b3e4f14b87338.png)

### 3. 确认玩家名和版本一致

启动前请确认：

- 所有玩家游戏版本一致
- 所有玩家模组列表一致
- 玩家名称不要重名，便于联机时区分

![启动前检查](/help-img/f3f9587530f44ef48d36065c22ca5941.png)

## 三、Java 版联机步骤

### 1. 房主开启世界

房主进入单人游戏后，选择已有存档或新建世界。

![选择世界](/help-img/cc4371ce9c5b469d9fc25be4fe08afe1.png)

### 2. 对局域网开放

进入世界后按 `Esc` 打开菜单，点击 **对局域网开放**。

如果你使用了支持更多选项的联机 Mod，通常还能设置：

- 在线模式 / 离线模式
- 端口号
- 是否允许作弊

如果好友不是通过正版会话进入，通常需要将在线模式切换为离线模式，或者关闭正版验证后再开房。

![对局域网开放](/help-img/9b262e1dfae44bdc94514393ec1d383f.png)

### 3. 好友加入服务器

其他玩家进入 **多人游戏** 后，可以：

- 直接在列表中搜索到局域网房间后加入
- 或点击 **直接连接**，手动输入房主虚拟 IP

房主的虚拟 IP 可以在好朋友联机工具房间列表中右键复制。

![多人游戏连接](/help-img/9401ddcb12524752b484ee15eb616e2a.png)

如果房主没有用默认端口 `25565`，则要按下面格式输入：

`房主虚拟IP:端口`

例如：

`10.0.0.8:25570`

![自定义端口连接](/help-img/399e3a4e58874df79ca015b833faef77.png)

## 四、独立服务端联机方式

如果你开的不是单人存档，而是独立服务端，那么连接方式更直接：

1. 房主先在本机启动 Minecraft 服务端。
2. 默认端口通常为 `25565`。
3. 好友在多人游戏输入 `房主虚拟IP:25565` 即可。

这种方式更适合多人长期存档。

## 五、常见问题

### 1. 搜不到世界

如果好友看不到房间，优先检查：

- 双方是否进入同一个联机房间
- 房主是否真的点击了 **对局域网开放**
- 是否输入了房主的虚拟 IP，而不是 `127.0.0.1`
- 防火墙是否拦截了 Java、Minecraft 或联机工具

### 2. 登录会话无效

出现 **登录会话无效**，通常是因为当前联机方式仍在校验正版会话，而当前客户端没有通过该校验。

可以按下面两种办法处理。

#### 方案 A：使用更高级的局域网联机 Mod，关闭正版验证

1. 在启动器中安装支持更多联机选项的局域网联机 Mod。
2. 房主进入世界后，打开 **对局域网开放**。
3. 在联机设置里关闭 **在线模式** 或 **正版验证**。
4. 重新开放房间，让好友再次连接。

#### 方案 B：使用皮肤站登录，跳过正版登录限制

1. 在启动器中添加同一个皮肤站类型的账号系统。
2. 房主和好友都使用皮肤站账号登录对应实例。
3. 确保大家使用相同游戏版本、相同模组环境。
4. 重新进入房间并连接。

### 3. `invalid characters in username`

这个报错通常表示用户名格式不合法。

处理办法：

- 用户名只能使用 **英文字母和数字**
- 不要包含空格
- 不要包含中文
- 不要包含特殊符号，最稳妥的做法是只用英文和数字

例如：

- `Steve01`
- `Alex2025`
- `Player9`

### 4. 连得上工具，进不去游戏

这类问题一般和以下原因有关：

- 游戏版本不一致
- Mod 或整合包不一致
- 服务端与客户端要求不一致
- 房主网络不稳定

### 5. 延迟高、容易掉线

建议依次排查：

- 尽量使用稳定网络，避免后台下载
- 先等待联机工具延迟稳定后再进游戏
- 在工具里尝试更稳定的线路或中转模式
- 服务端玩家较多时，适当降低视距和实体负载

## 六、总结

我的世界通过好朋友联机工具联机时，最关键的是：

1. 先让所有人进入同一个联机房间。
2. 房主开放局域网或启动独立服务端。
3. 好友使用 **房主虚拟 IP + 端口** 连接。
4. 出现报错时，优先检查版本、模组、正版验证和用户名格式。
