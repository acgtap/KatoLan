# 泰拉瑞亚使用好朋友联机工具联机教程

![泰拉瑞亚联机封面](/help-img/tlry.png)

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **泰拉瑞亚（Terraria）**。

## 一、安装联机工具并进入同一房间

- 房主和好友都需要先安装好朋友联机工具（保证版本一致）。
- 进入软件后，由一名玩家创建房间，其他玩家加入同一个房间。
- 看到房间成员列表后，等待延迟稳定再进入游戏。

![联机工具主页](/help-img/rjsy.png)

![房间列表](/help-img/hylb.png)

## 二、房主创建泰拉瑞亚世界

进入游戏后，由一名玩家选择：

1. 点击`多人模式`
2. 点击`开服并开始游戏`
3. 选择角色和世界
4. 设置服务器密码，也可以留空

完成后，这名玩家就是游戏房主。

![创建多人世界](/help-img/ae2d945f79b9447facf08c67b23d9b6d.png)

## 三、好友加入游戏

好友进入泰拉瑞亚后：

1. 点击 `多人模式`
2. 选择 `加入`
3. 选择角色
4. 输入房主的虚拟 IP

房主的虚拟 IP 可以在联机工具房间列表中右键复制。

![输入房主 IP](/help-img/338ec78e3c62484ab6f975cbfa9b32f4.png)

## 四、输入默认端口和密码

泰拉瑞亚本地服务器默认端口通常为 `7777`。

- 如果房主没有改端口，直接输入 `7777`
- 如果房主设置了密码，还需要再输入密码才能进入

![输入端口](/help-img/f5ddf6f0f88044079b7008af22add59c.png)

![输入密码](/help-img/96d47cf51d294ada897e8aa9ed4fe45d.png)

## 五、开始联机

全部设置完成后，就可以和好友一起探索、挖矿、建家和挑战 Boss。

![联机成功](/help-img/78a48169315b4ef19edbc28d44584c24.png)

## 六、常见问题

### 1. 连不上房主

优先检查：

- 是否已经在同一个联机房间
- 输入的是不是房主虚拟 IP
- 端口是否为 `7777`
- 防火墙是否拦截 Terraria

### 2. 加入后卡住或掉线

建议：

- 等联机工具延迟稳定后再进游戏
- 关闭后台下载和占带宽程序
- 尝试更稳定的线路或中转模式
- 使用增强模式，设置中开启禁用 P2P 和 kcp 代理

![增强设置](/help-img/增强模式设置.png)

## 七、总结

泰拉瑞亚联机最核心的步骤就是：

1. 双方先进入同一个联机房间（输入相同的房间名）。
2. 房主创建多人世界。
3. 好友输入房主虚拟 IP 和 `7777` 端口加入。
