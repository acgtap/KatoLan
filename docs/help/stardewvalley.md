# 星露谷物语使用好朋友联机工具联机教程

![星露谷联机封面](/help-img/xlgwy.png)

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **星露谷物语（Stardew Valley）**。

## 一、先进入同一个联机房间

- 房主和好友都先打开好朋友联机工具。
- 由一名玩家创建房间，其他玩家加入同一个房间。
- 等房间内延迟稳定后再进游戏。

![联机工具主页](/help-img/rjsy.png)

![房间列表](/help-img/hylb.png)

## 二、房主创建合作农场

启动游戏后，由房主执行：

1. 进入 `合作模式`
2. 选择 `主持新农场`
3. 设置角色名、农场名和基础选项
4. 点击 `OK` 创建农场

![主持新农场](/help-img/cc69eba0efd44ac58bc375e1e77b81c1.png)

## 三、开启接受 IP 联机

房主进入农场后，按 `Esc` 打开设置，找到 **多玩家设置**。

这里必须开启 **接受 IP 联机**，否则好友即使知道 IP 也无法进入。

![开启接受 IP 联机](/help-img/d52768b341ca4a2c8bd31a2e93f5c6dd.png)

## 四、好友通过 IP 加入

好友进入游戏后：

1. 进入 `合作模式`
2. 点击 `加入局域网游戏`
3. 输入房主的虚拟 IP

房主的虚拟 IP 可以在联机工具房间里右键复制。

![输入 IP](/help-img/524e71447ed1468eace90654b29cb0d6.png)

![复制房主 IP](/help-img/6dc908c636f543749f81c9a3d30d0152.png)

确认后会弹出房主创建好的农场，点击即可加入。

![加入房间](/help-img/b8fb2ac925334bd4a918078572a3dd81.png)

![联机成功](/help-img/70af2be8992d4c69b0aa4aa547ce03ec.png)

## 五、常见问题

### 1. 输入 IP 后搜不到农场

- 确认房主已经开启 `接受 IP 联机`
- 确认输入的是房主虚拟 IP
- 确认双方已进入同一个联机房间

### 2. 进房失败

- 检查双方游戏版本是否一致
- 如果装了 Mod，确认模组环境一致
- 临时关闭防火墙再测试一次

## 六、总结

星露谷物语联机最关键的是：

1. 先进入同一个联机房间。
2. 房主创建合作农场并开启 `接受 IP 联机`。
3. 好友通过房主虚拟 IP 加入。
