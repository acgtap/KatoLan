## 使用前准备

在开始联机前，请先确认以下几点：

1. 已安装并更新到最新版好朋友联机工具。
2. 你和好友使用的是同一款游戏版本。
3. 已临时关闭防火墙、杀毒软件或将联机工具加入白名单。
4. 双方都已加入同一个联机房间。
