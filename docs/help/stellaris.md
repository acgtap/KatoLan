# 群星使用好朋友联机工具联机教程

![群星联机封面](/help-img/qx.png)

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **群星（Stellaris）**。

## 一、下载并安装联机工具

先安装好朋友联机工具。安装过程中如果网卡驱动安装失败，建议使用管理员身份启动软件再试一次。

![启动联机工具](/help-img/rjsy.png)

## 二、创建和加入房间

联机时：

- 一名玩家作为房主，选择延迟较低的节点后创建房间
- 其他玩家输入相同房间名加入

房间里会显示所有成员的虚拟 IP 和延迟信息。

![创建房间](/help-img/cjfj.png)

![成员列表](/help-img/hylb.png)

## 三、房主创建群星主机

当所有好友都进入房间后，启动群星：

1. 进入 `多人游戏`
2. 点击 `建立主机`
3. 输入服务器名称
4. 创建游戏房间

![多人游戏](/help-img/7ea86a20a4414ce3bf1953293023d1c9.png)

![建立主机](/help-img/af8e56c50ec14cc1a7056a4c324d25b7.png)

## 四、其他玩家加入房间

其余玩家进入多人游戏后：

- 搜索服务器
- 找到房主建立的房间
- 点击加入游戏

![搜索并加入](/help-img/240afcfa379247f1aeeeae1f78445f3c.png)

进入大厅后选择自己的帝国并准备，就可以开始联机。

![选择帝国](/help-img/e7acb51f2efe4cd8a64d02e8749eea33.png)

## 五、联机前必须确认

群星联机对版本一致性要求比较高，建议提前确认：

- 游戏版本一致
- DLC 状态一致
- Mod 列表和顺序一致
- 如果你使用的是学习版，必须添加一个群星的联机补丁才能正常联机
- 如果是正版游戏，但是搜索不到房主创建的游戏或者加入不了也需要打一个联机补丁。
- 联机补丁下载地址：[cngame.wiki](https://wp.cngame.wiki/)

不一致时通常会导致搜不到房间、加入失败或进房后不同步。

## 六、总结

群星通过好朋友联机工具联机时，核心流程是：

1. 先用联机工具进入同一个房间。
2. 房主在群星内建立主机。
3. 其他玩家搜索服务器加入。
4. 保证版本、DLC、Mod 一致。
