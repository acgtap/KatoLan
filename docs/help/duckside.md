# 逃离鸭科夫使用好朋友联机工具联机教程

注意：联机前建议所有玩家都先通过新手教程，并且必须由 **房主先进入**，然后 **客机再进入**，否则容易出现连接异常。

## 一、先安装联机 Mod

进入 Steam 创意工坊，下载 **逃离鸭科夫 / Duckside** 对应的联机 Mod。

文中说明是：**mod 里面全都要勾选**，并且每个联机玩家都需要安装相同内容、按同样顺序启用。

![下载联机 Mod](/help-img/68f900b844ec1.png)

![创意工坊页面](/help-img/68f90122ba4bd.png)

按照教程中的顺序勾选并启用，确保所有人配置一致。

![勾选并启用 Mod](/help-img/68f90110ebc42.png)

## 二、房主操作

### 1. 切换到服务器模式

安装完成后，会先出现一个弹窗。

![安装后弹窗](/help-img/68f9015ae42bb.png)

这里需要点击切换到 **服务器模式**。

![切换到服务器模式](/help-img/68f901733d937.png)

### 2. 房主进入游戏

切换完成后，由房主先进入游戏。

![房主进入游戏](/help-img/68f901942ad99.png)

### 3. 创建 KatoLan 房间

进入游戏后，再打开 **好朋友联机工具（KatoLan）**，由房主创建一个房间，让朋友也一起加入进来。

![创建 KatoLan 房间](/help-img/68f9020f6607c.png)

完成后，房主端的准备工作就基本完成了。

## 三、朋友如何进入

### 1. 先进入同一个 KatoLan 房间

朋友也要先进入和房主相同的 KatoLan 房间，确保双方已经处于同一个虚拟局域网环境中。

### 2. 启动游戏并选择加入方式

朋友启动游戏后，也会看到对应的连接弹窗。

![加入弹窗](/help-img/68f9028789cdc.png)

### 3. 输入房主 IP 手动连接

在该界面中输入 **房主的虚拟 IP 地址**，然后点击 **手动连接**。

房主的虚拟 IP 可以在 KatoLan 房间成员列表中查看和复制。

显示连接完成后，就可以继续进入游戏。

![手动连接完成](/help-img/68f902acd6f8b.png)

## 四、联机要点

为了减少失败概率，建议严格按下面顺序操作：

1. 所有人都先完成新手教程。
2. 所有人安装相同的联机 Mod，并按相同顺序启用。
3. 房主先切换到服务器模式并先进游戏。
4. 再让朋友加入 KatoLan 房间并启动游戏。
5. 朋友通过房主虚拟 IP 手动连接。

## 五、常见问题

### 1. 连接失败

优先检查：

- 房主是否先进入游戏
- 双方是否已进入同一个 KatoLan 房间
- 输入的是否是房主虚拟 IP
- 双方联机 Mod 是否完全一致

### 2. 看不到联机选项或无法连上

通常和以下原因有关：

- Mod 没有全部勾选
- 启用顺序不一致
- 房主没有切换到服务器模式
- 一方还没过新手教程

## 六、总结

逃离鸭科夫通过好朋友联机工具联机时，最关键的是：

1. 所有人先装好并启用一致的联机 Mod。
2. 房主切换到服务器模式并先进游戏。
3. 所有人进入同一个 KatoLan 房间。
4. 客机通过房主虚拟 IP 手动连接。
