# 饥荒联机版使用好朋友联机工具联机教程

![饥荒联机封面](/help-img/jh.png)

本文介绍如何使用 **好朋友联机工具（KatoLan）** 和好友远程联机游玩 **饥荒联机版（Don't Starve Together）**。

## 一、先进入同一个联机房间

房主和好友都先打开好朋友联机工具：

- 一名玩家创建房间
- 其他玩家输入相同房间号加入
- 房间内显示成员信息后，等待延迟稳定

![联机工具主页](/help-img/rjsy.png)

![房间列表](/help-img/hylb.png)

### 重要说明：建议使用离线模式启动游戏

使用这种联机方式时，建议房主和加入者都在启动 **饥荒联机版** 时选择 **离线模式**。

原因是如果使用在线模式，游戏内的房间列表有时不会正常显示房主创建的房间，导致好友在 `浏览游戏` 里搜不到。

为了避免房间不显示，最稳妥的做法是：

- 启动游戏时直接选择 **离线模式**
- 房主和加入者都保持同样的启动方式
- 进入游戏前先确认已经加入同一个好朋友联机工具房间

## 二、房主创建游戏

启动游戏后，由一名玩家作为房主：

1. 点击 `创建游戏`
2. 输入房间名称
3. 选择 `仅本地`
4. 设置玩家人数
5. 点击生成世界

![创建游戏](/help-img/248b0324263241129529aad8da6a2129.png)

## 三、其他玩家加入

好友进入游戏后：

1. 点击 `浏览游戏`
2. 刷新列表
3. 找到房主创建的房间
4. 点击加入

![浏览游戏](/help-img/0ec5c8f8f5ab420ca4d2ada18dfd825f.png)

进入后，所有玩家再各自选择角色即可开始游戏。

![选择角色](/help-img/934b4f2cf2a6453fbbdffef501c80bbf.png)

## 四、使用命令直接连接房间

如果你在 `浏览游戏` 里看不到房主创建的房间，也可以尝试使用命令方式直接连接。

这种方式更适合以下情况：

- 房间列表刷新不出来
- 已经确认双方在同一个 KatoLan 房间里
- 使用离线模式后仍然搜不到房间

### 1. 先获取房主的虚拟 IP

回到好朋友联机工具界面，在房间成员列表中找到房主，复制房主的虚拟 IP。

### 2. 进入游戏控制台

进入饥荒联机版后，打开控制台或命令输入界面。

如果你的游戏默认没有开启控制台，需要先在游戏设置或相关配置中启用控制台功能。

### 3. 输入连接命令

在控制台中使用直连命令连接房主。

常见写法如下：

```lua
c_connect("房主虚拟IP", 10999)
```

例如：

```lua
c_connect("10.0.0.8", 10999)
```

其中：

- `房主虚拟IP` 改成房主在 KatoLan 房间中的虚拟 IP
- `10999` 是饥荒联机版常见的联机端口，若房主修改过端口，需要换成实际端口

### 4. 命令连接时的注意事项

- 房主必须已经创建好世界并进入游戏
- 房主和加入者建议都使用 **离线模式** 启动游戏
- 双方游戏版本、模组、世界设置仍然要保持一致
- 如果输入命令后没有反应，优先检查 IP 是否复制正确，以及防火墙是否拦截游戏

## 五、检查联机质量

全部玩家进入后，可以在设置里查看玩家信息和主机性能。

如果主机性能显示良好，通常就可以比较流畅地联机。

![查看玩家性能](/help-img/1967fa39a84443db9384c4b5fcf6da18.png)

## 六、常见问题

### 1. 搜不到房间

- 确认双方已经进入同一个联机房间
- 让房主重新创建世界后再刷新
- 检查是否选择了 `仅本地`

### 2. 游戏卡顿

- 等联机工具延迟稳定后再进游戏
- 房主尽量关闭占用带宽的软件
- 尝试使用更稳定的线路

## 七、总结

饥荒联机版通过好朋友联机工具联机时，关键步骤是：

1. 先进入同一个联机房间。
2. 房主创建 `仅本地` 世界。
3. 好友通过 `浏览游戏` 搜索并加入。
